| `cv-variant-03-retail-analytics-es.tex`  | Spanish, Falabella/data-science-forward  | More emphasis on analytics, allocation, replenishment, and model testing |

| `cv-variant-05-academic-polished-es.tex` | Spanish, polished academic/current style | A refined version closest to the current visual identity                 |

Compile with `tectonic <file>.tex` locally, or upload any file to Overleaf. These candidates avoid `fontawesome5` so local compilation is more reliable than the current icon-based CV source.
